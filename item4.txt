Black box a soft testing method in wich we don't know the code we don't have acces to it, we just test the functionality and features of the application, and behavior of the soft, aplicable to higher levels of testing of a software, it is very small time consuming, as an example check if it works to log in on a page.
White box testing is a testing method where the code is known to us, usually done by developers, but made by testers too, it is mandatory to have programming skills, it is very time consuming, it is internal software testing.Example, testing a loop within a code, or a branch of a if statement if it returns the desired output.
